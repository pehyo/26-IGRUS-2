# 26-IGRUS-픅젝대회 추가부문
1. 실행 방법

- step 1. 기본적으로 설정되어있는 체크리스트 목록이 나옴.
  1-1. 체크리스트 목록 이름 수정을 누르면 원하는 이름으로 수정가능함
  1-2. 목록삭제도 가능함
- step 2. 원하는 목록을 클릭한 뒤 '할 일 입력' 칸에 할 일을 입력하고 추가버튼 누르면 체크리스트 추가완료
  2-1. 할 일 입력 옆 날짜 지정을 통해 마감 날짜 수행 가능
  2-2. 드래그를 통해 할 일 순서 바꾸기 가능
  2-3. 삭제를 눌러 할 일 삭제 가능
  
- step 3. 목록 화면에서 해당 목록의 이행도를 확인할 수 있음.

  
======






2. 작품소개

체크리스트 광인이 온라인으로도 체크리스트가 필요하여 만들어보았습니다. 메모장을 이용해서 사용했는데 이행도 같은 부분이 없어서 쾌감과 동기가 부족하다 느껴 이를 보완하여 체크리스트를 만들어보았습니다.
나갔다 들어와도 초기화 되지 않습니다! 쇼핑리스트, 버킷리스트, 할 일 등으로 나눠서 마음껏 써보세요!
